Web Decoder

Basic instructions: Right click on some text and select "decode" from the popup menu. Note: If the decoder is not processing enough of the text, for example, with css color functions, it may be necessary to select the text before decoding to ensure the correct text is used. The decoder will show a dialog at the bottom of the screen providing an explanation of the text you selected. You can even use it to decode the examples below.

Current decoder capabilities:

  • base64 encoded text. SGVsbG8sIERlY29kZXIhCg==, eyJzb21lIjogImpzb24ifQo=
  • css colors. #FF00AA, rgb(128, 128, 200), aquamarine
  • unix timestamps (seconds or millisecons since epoch). 88930800, 978336000
  • more on the way
